================================================================================
ART REFERENCE: CHARACTERS - DUNOON 2056

This document contains all required character art assets.

================================================================================
PLAYER CHARACTER - KUNDAI
================================================================================

The player character's appearance is partially determined by player choice.
Base design must accommodate variations.

KUNDAI - BASE DESIGN
---------------------
Age: Early twenties (20-25)
Ethnicity: Black African (variable ancestry)
Ancestry options:
  - Malawian (sharper features, warm brown skin)
  - Nigerian (broader features, darker skin)
  - Mozambican (softer features, medium brown skin)
  - Zimbabwean (angular features, warm brown skin)

GENDER VARIANTS
---------------

VARIANT A: Masculine (Man)
  - Gender: Male
  - Build: Lean, wiry, survival-hardened
  - Height: Average to tall
  - Face: Sharp, angular features
  - Hair: Close-cropped, practical
  - Eyes: Hard, haunted, like they've seen too much
  - Distinguishing marks:
    * Scar running up left forearm (old, healed)
    * Tattoo on right bicep: numbers/date, partially obscured by burn marks
    * Neural implant port at base of skull (visible when hair moved)

VARIANT B: Feminine (Woman)
  - Gender: Female
  - Build: Lean, wiry, survival-hardened
  - Height: Average
  - Face: Strong, masculine features, hard jaw
  - Hair: Braided close to skull
  - Eyes: Hard, haunted, like they've seen too much
  - Distinguishing marks:
    * Scar running up left forearm (old, healed)
    * Tattoo on right bicep: numbers/date, partially obscured by burn marks
    * Neural implant port at base of skull (visible when hair moved)

KUNDAI - EXPRESSIONS REQUIRED
-----------------------------
1. Confused/Amnesiac (default early game)
2. Determined/Focused
3. Pain/Implant headache
4. Hope/Remembering
5. Sadness/Loss
6. Angry/Vengeful
7. Peaceful/Accepting
8. Glitching/Implant malfunction (overlay effect)

KUNDAI - POSES REQUIRED
-----------------------
1. Standing, neutral
2. Sitting on mattress
3. Using neural deck
4. Examining photograph
5. Looking in broken mirror
6. Walking through township
7. Standing at beach, facing ocean
8. Unconscious/Fallen (for memory sequences)

KUNDAI - OUTFITS
----------------
Default:
  - Worn t-shirt (faded color)
  - Cargo pants (multiple pockets, worn)
  - Boots that have seen better days

Optional (purchasable):
  - Armored jacket (hidden reinforcement)
  - Dark clothes (nondescript, blending)
  - Work boots (sturdier)

================================================================================
INTERNAL VOICES (ABSTRACT/VISUAL REPRESENTATIONS)
================================================================================

These represent internal psychological aspects. They should have a ghostly,
internal quality - not fully present, but suggestions.

THE HACKER
----------
Concept: Voice of technology, systems, and digital knowledge
Visual Style: Digital, electric, pattern-seeing

Appearance:
  - Abstract representation, not literal person
  - Electric blue/cyan color scheme
  - Circuit pattern overlays
  - Data stream visualizations around them
  - Eyes that see code/patterns
  - Gender: Ambiguous (aspect of self)
  - Expression: Analytical, curious, calculating

Expressions:
  1. Analytical (examining data)
  2. Excited (found a pattern)
  3. Frustrated (blocked by ICE)
  4. Triumphant (hack succeeded)

Visual Effects:
  - Glitch overlay when speaking
  - Code scrolling in background
  - Neural connection visualization

THE SURVIVOR
------------
Concept: Voice of instinct, street smarts, and self-preservation
Visual Style: Gritty, scarred, always watching

Appearance:
  - Abstract representation
  - Muted earth tones, gray, red accents
  - Scars visible (symbolic of past trauma)
  - Eyes always scanning
  - Tense posture
  - Gender: Ambiguous (aspect of self)
  - Expression: Wary, calculating, protective

Expressions:
  1. Wary (sensing danger)
  2. Confident (escape route found)
  3. Paranoid (trust no one)
  4. Relieved (safe for now)

Visual Effects:
  - Red threat highlights when danger detected
  - Escape route visualization
  - Shadow/stealth overlay

THE ANCESTOR
------------
Concept: Voice of wisdom, heritage, and spiritual connection
Visual Style: Ancient, warm, connected, timeless

Appearance:
  - Abstract representation
  - Warm gold, amber, deep brown colors
  - Traditional African patterns/textiles suggested
  - Eyes that hold generations of wisdom
  - Peaceful, patient presence
  - Gender: Ambiguous (collective of ancestors)
  - Expression: Patient, wise, compassionate

Expressions:
  1. Patient (waiting, listening)
  2. Compassionate (understanding pain)
  3. Stern (moral judgment)
  4. Proud (achievement recognized)

Visual Effects:
  - Ancestral silhouettes behind
  - Warm glow
  - Traditional patterns overlay
  - Memory fragments visible

VOICES UNIFIED
--------------
Special moment when all voices harmonize:
  - All three visual styles merged
  - Balanced, centered
  - Powerful, complete

TANGO
-----
Role: Tech fixer, implant specialist, information broker
First Appearance: After Mama Nomi recommendation

Physical Description:
  - Gender: Male
  - Age: Late 20s to early 30s
  - Build: Lean, wiry, focused intensity
  - Face: Angular, intelligent, augmented
  - Hair: Short, practical, maybe undercut
  - Eyes: Augmented - faint glow visible, optical implants
  - Skin: Medium brown
  - Distinguishing features:
    * Optical implant glow (faint blue/green)
    * Cybernetic tool fingers on one hand (precision work)
    * Tech tattoos (circuit patterns)
    * Always has tools nearby

Clothing:
  - Practical workshop clothes
  - Tool belt
  - Fingerless gloves (for precision work)
  - Tech goggles pushed up on forehead
  - Stained with solder/electronics work

Expressions:
  1. Focused (working)
  2. Skeptical (assessing player)
  3. Impressed (seeing interesting tech)
  4. Concerned (implant diagnosis)
  5. Secretive (knowing more than saying)

Poses:
  1. Hunched over workbench
  2. Examining player's implant
  3. Arms crossed, skeptical
  4. Showing equipment for sale
